# 《阡陌云图》“挑战杯”5分钟路演答辩稿

**【第1页：封面页】** *(预计时长：10秒)*

尊敬的各位评委老师，大家好！我是来自西南科技大学的胡安雷。今天，我代表团队带来的参赛项目是《阡陌云图——乡村治理智慧决策支持系统》。我们致力于用AI，守护每一寸乡土。

**【第2页：项目缘起】** *(预计时长：20秒)*

项目的缘起，要从我寒假返乡时看到的一幕说起。当时，一位村支书因为听不懂老人的方言，错把“低保”办成了“医保”。这让我深刻意识到：横亘在乡村老人面前的数字鸿沟，本质上不是“没有网”，而是“交互不通”。

老人们面对复杂的手机软件无从下手。从那时起我们决定，要用“语言交互”代替传统的“点击屏幕”，为乡村治理蹚出一条新路。

**【第3页：行业背景】** *(预计时长：15秒)*

当前，我国农村老龄化正在加速。与此同时，国家多部门联合印发了《数字乡村发展工作要点》，明确要求夯实乡村数字基建。这为我们的项目提供了极其坚实的政策东风与广阔的下沉市场空间。

**【第4页：实地调研】** *(预计时长：15秒)*

为了验证痛点，我们团队历经300多公里，走访川滇两省，收集了百余份真实问卷。我们发现，当前的数字乡村建设普遍存在三大问题：重建设轻运营、老人操作门槛高、基层干部面临严重的数据孤岛。

**【第5页：项目简介】** *(预计时长：15秒)*

基于这些痛点，我们融合了前沿的大语言模型技术，打造了国内首个乡村治理智慧决策中枢——阡陌云图。它具备三大核心能力：听得懂村民的方言、算得准业务的流转、管得好全村的数据。

**【第6页：村民端】** *(预计时长：20秒)*

首先是村民端，这是专为留守老人打造的“数字拐杖”。老人完全不需要识字或打字，只需按住按钮，用方言说一句“家里灯坏了”，系统就能以98%的极高准确率识别意图，并自动填写生成报修工单。全流程都有语音播报，让办事就像“拉家常”一样简单。

**【第7页：干部端】** *(预计时长：20秒)*

其次是干部端，这是村两委的“全天候数字助手”。村民语音上报后，AI会自动派单给对应的网格员。更关键的是，村干部只需下达一句语音指令，系统就能调用本地台账，秒级生成规范的防汛排查、纠纷调解等红头文件，真正把基层干部从繁杂的表格中解放出来。

**【第8页：决策端】** *(预计时长：15秒)*

最后是决策端。我们构建了全域感知的“数字驾驶舱”。将村里的耕地、灾害隐患点、独居老人等信息在三维地图上实时可视。一旦出现异常，系统毫秒级报警，让乡村治理从“被动救灾”转变为“主动防范”。

**【第9页：技术壁垒】** *(预计时长：35秒)*

各位评委可能会问，通用大模型这么多，我们的壁垒在哪里？我们筑起了四道极其坚固的护城河：

第一，**听得懂土话**。我们自建了500多个小时的方言语料库并进行微调，攻克了农村环境杂音大、口音重的识别难题。

第二，**懂政策**。我们建立了本地政策知识库，让AI的每一次回答都基于真实文件，彻底杜绝了虚假捏造。

第三，**能干活**。首创多智能体协同，实现了业务的无人干预闭环。

第四，**守秘密**。这是最重要的一点，我们将大模型压缩部署在村委会的本地电脑上，敏感数据“绝对不出村”，从根本上打消了政府的安全顾虑。

**【第10页：商业模式】** *(预计时长：25秒)*

在商业落地方面，我们跑通了“G-B-C 三轮驱动”生态。

在G端，我们向政府提供系统并收取极具性价比的SaaS订阅费；在C端，坚持对村民免费，培养高频使用粘性；在B端，我们在严格脱敏的前提下，将沉淀的涉农数据、信用资产对接给农业金融机构或农资企业，实现长效的商业变现闭环。

**【第11页：试点反馈】** *(预计时长：25秒)*

目前，系统已在两省6县进行实地试点。数据证明：**系统让工单处理效率提升了70%，群众诉求响应时间从过去的48小时缩短至2小时！** 屏幕上展示的，正是试点村委会为我们联合署名出具的《试用反馈函》，这是对我们实战价值最大的肯定。

**【第12页：项目价值】** *(预计时长：15秒)*

阡陌云图不仅是一套系统，它更在孵化数字新农人、重塑基层治理的温度，致力于定义中国数字乡村建设的新标准。

**【第13页 - 第14页：团队成员与专家】** *(预计时长：20秒)*

硬核的产品离不开优秀的团队。我们是一支涵盖了软件工程、信息安全、财务管理等多学科交叉的实干铁军。同时，项目得到了多位省部级专家导师的倾力指导，为技术的合规与落地保驾护航。

**【第15页：团队知识产权】** *(预计时长：15秒)*

凭借扎实的研发，团队目前已斩获5项国家计算机软件著作权，并在中国大学生计算机设计大赛等众多国家级、省级竞赛中屡获殊荣，构筑了坚实的知识产权壁垒。

**【第16页：成长潜力与融资计划】** *(预计时长：20秒)*

**[语气提示：充满信心与展望]**

在财务预测方面，随着SaaS模式在下沉市场的快速铺开，预计到第三年，我们的营业收入将突破1000万元。本次我们计划寻求种子轮融资50万元，出让10%的股权。资金将核心用于方言模型的深度微调与川滇市场的推广铺设。

**【第17页：发展规划】** *(预计时长：10秒)*

未来，我们将立足四川打响品牌，逐步开拓中西部市场，最终走向全国，甚至将适老化的数字经验辐射至东盟国家。

**【第18页：结束语】** *(预计时长：10秒)*

**[动作提示：站定，目光真诚，微鞠躬]**

科技的魅力不仅在于前沿的突破，更在于它能平等地照亮每一个被遗忘的角落。阡陌云图，愿用AI守护每一寸乡土。我的汇报完毕，谢谢各位评委老师！
