## 1. 项目概述

### 1.1 项目背景

当前肿瘤筛查存在以下痛点：
- **效率低下**: 传统筛查依赖人工，耗时长
- **准确性不足**: 依赖医师经验，标准不统一
- **覆盖面有限**: 医疗资源分布不均，偏远地区难以获得优质筛查服务
- **成本高昂**: 单次筛查成本高，难以大规模推广

### 1.2 项目定位

本系统是一个基于人工智能的肿瘤风险评估平台，通过：
- 智能问卷采集用户健康数据
- 机器学习算法评估肿瘤风险
- 可视化报告展示评估结果
- 个性化健康建议推荐

实现**早发现、早诊断、早治疗**的癌症防控目标。

### 1.3 项目范围

**包含的功能**:
- ✅ 用户注册登录系统
- ✅ 智能健康问卷
- ✅ 风险智能评估
- ✅ 可视化风险报告
- ✅ 历史记录管理
- ✅ 管理后台
- ⭐ 医学影像上传分析（可选）
- ⭐ 知识图谱展示（可选）

**不包含的功能**:
- ❌ 实际医疗诊断（仅提供风险评估参考）
- ❌ 在线问诊
- ❌ 药品推荐
- ❌ 医疗支付

### 1.4 技术架构

- **系统类型**: B/S架构的Web应用
- **前端技术**: React 18 + Ant Design 5
- **后端技术**: Flask/FastAPI + PostgreSQL
- **AI技术**: XGBoost + SHAP + 深度学习（可选）
- **部署方式**: Docker容器化部署

## 4. 功能需求

### 4.1 用户认证模块（AUTH）

#### AUTH-001: 用户注册

**优先级**: P0（必须）

**功能描述**:  
新用户通过填写基本信息完成账号注册

**输入**:
- 手机号/邮箱（账号）
- 密码（8-20位，含字母数字）
- 验证码（图形验证码）
- 用户协议确认

**处理逻辑**:
1. 验证输入格式
2. 检查账号是否已存在
3. 密码加密存储（bcrypt）
4. 生成用户UUID
5. 发送注册成功通知

**输出**:
- 注册成功提示
- 自动登录并跳转首页

**异常处理**:
- 账号已存在 → 提示"该账号已注册"
- 验证码错误 → 提示"验证码错误，请重新输入"
- 密码强度不足 → 提示密码要求

#### AUTH-002: 用户登录

**优先级**: P0（必须）

**功能描述**:  
已注册用户通过账号密码登录系统

**输入**:
- 账号（手机号/邮箱）
- 密码
- 记住我（可选）

**处理逻辑**:
1. 验证账号密码
2. 生成JWT Token（有效期7天）
3. 记录登录日志
4. 返回用户基本信息

**输出**:
- JWT Token（存储在localStorage）
- 跳转至首页/上次访问页面

**异常处理**:
- 账号不存在 → 提示"账号或密码错误"
- 密码错误 → 提示"账号或密码错误"
- 账号被禁用 → 提示"账号已被冻结，请联系管理员"

#### AUTH-003: 找回密码

**优先级**: P1（应该）

**功能描述**:  
用户忘记密码时通过手机/邮箱重置密码

**输入**:
- 注册手机号/邮箱
- 验证码（短信/邮件）
- 新密码

**处理逻辑**:
1. 验证账号存在性
2. 发送验证码（有效期5分钟）
3. 验证码校验
4. 更新密码（加密存储）

**输出**:
- 密码重置成功提示
- 跳转至登录页

### 4.2 智能问卷模块（QUESTIONNAIRE）

#### QUEST-001: 多步骤问卷填写

**优先级**: P0（必须）

**功能描述**:  
通过4步问卷采集用户健康数据

**问卷结构**:

**第1步：基本信息**
```
字段清单:
1. 姓名: 文本输入（可选，用于报告个性化）
2. 年龄: 数字输入（18-120）【必填】
3. 性别: 单选（男/女）【必填】
4. 身高: 数字输入（cm, 100-250）【必填】
5. 体重: 数字输入（kg, 30-200）【必填】
6. BMI: 自动计算（体重/身高²）
7. 民族: 单选（汉族/少数民族）
8. 职业: 下拉选择（20+职业选项）
```

**第2步：生活习惯**
```
字段清单:
1. 吸烟史: 单选
   - 从不吸烟
   - 已戒烟（戒烟年限: ___ 年）
   - 偶尔吸烟（每周<3支）
   - 每天吸烟（支数: ___ 支/天，烟龄: ___ 年）

2. 饮酒史: 单选
   - 从不饮酒
   - 偶尔饮酒（社交场合）
   - 每周饮酒（频率: ___ 次/周）
   - 每天饮酒（类型: 白酒/啤酒/红酒）

3. 运动习惯: 单选
   - 几乎不运动
   - 偶尔运动（每月1-2次）
   - 每周运动1-2次
   - 每周运动3次以上

4. 饮食习惯: 多选
   - 经常吃腌制食品
   - 偏好油炸食品
   - 爱吃烧烤
   - 高盐饮食
   - 饮食均衡

5. 睡眠质量: 单选
   - 良好（每天7-8小时）
   - 一般（6-7小时或8-9小时）
   - 较差（<6小时或>9小时）
   - 失眠
```

**第3步：疾病史与家族史**
```
字段清单:
1. 慢性病史: 多选
   - 高血压
   - 糖尿病
   - 冠心病
   - 慢性肝炎（类型: 乙肝/丙肝）
   - 慢性胃炎/胃溃疡
   - 无

2. 家族肿瘤史: 多选（可选择多个）
   每项包含: 患病亲属（父母/兄弟姐妹/祖父母/外祖父母）
   - 肺癌
   - 胃癌
   - 肝癌
   - 结直肠癌
   - 乳腺癌
   - 食管癌
   - 胰腺癌
   - 其他（文本输入）
   - 无家族肿瘤史

3. 手术史: 单选
   - 无
   - 有（手术名称: ___, 时间: ___）

4. 长期用药: 多选
   - 降压药
   - 降糖药
   - 免疫抑制剂
   - 激素类药物
   - 其他（文本输入）
   - 无
```

**第4步：症状自查**
```
字段清单:
1. 近3个月出现的症状: 多选
   - 持续咳嗽（>2周）
   - 咯血
   - 胸痛/胸闷
   - 不明原因体重下降（>5kg/月）
   - 持续乏力
   - 不明原因发热
   - 吞咽困难
   - 声音嘶哑
   - 淋巴结肿大
   - 皮肤黄疸
   - 大便带血/黑便
   - 腹痛/腹胀
   - 乳房肿块
   - 无症状

2. 既往检查异常: 多选
   - 肺部结节
   - 肝脏结节
   - 胃肠息肉
   - 甲状腺结节
   - 乳腺结节
   - 其他（文本输入）
   - 无异常

3. 上次体检时间: 单选
   - 3个月内
   - 3-6个月
   - 6-12个月
   - 1年以上
   - 从未体检
```

**交互特性**:
1. **实时验证**: 填写完每个字段自动验证
2. **条件显示**: 根据选择动态显示子问题
   - 例: 选择"每天吸烟" → 显示"支数"和"烟龄"输入框
3. **进度保存**: 每步完成自动保存到localStorage
4. **进度提示**: 顶部显示进度条（1/4, 2/4, 3/4, 4/4）
5. **错误提示**: 必填项未填写时红色高亮+提示文字

**处理逻辑**:
1. 前端实时验证用户输入
2. 每步完成后保存到本地存储（防止刷新丢失）
3. 最后一步提交时发送完整数据到后端
4. 后端验证数据完整性和合法性
5. 数据存储到数据库

**输出**:
- 跳转至"评估中"页面
- 显示加载动画和进度提示

#### QUEST-002: 问卷数据校验

**优先级**: P0（必须）

**功能描述**:  
对用户填写的问卷数据进行合法性和合理性校验

**校验规则**:

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 年龄 | 18 ≤ age ≤ 120 | "年龄应在18-120岁之间" |
| 身高 | 100 ≤ height ≤ 250 | "身高应在100-250cm之间" |
| 体重 | 30 ≤ weight ≤ 200 | "体重应在30-200kg之间" |
| BMI | 自动计算，异常提示 | BMI<15或>40时警告"数值异常，请核对身高体重" |
| 吸烟年限 | ≤ (年龄-10) | "吸烟年限不能超过实际可能年限" |
| 必填项 | 不能为空 | "请填写此必填项" |

**后端二次校验**:
- 检查数据类型
- 检查数值范围
- 检查逻辑一致性（如戒烟年限<总烟龄）

### 4.4 风险评估模块（ASSESSMENT）

#### ASSESS-001: 风险智能评估

**优先级**: P0（必须）

**功能描述**:  
基于问卷数据和机器学习模型，评估用户的肿瘤风险

**输入数据**:
- 问卷原始数据（JSON格式）
- 用户ID
- 评估时间戳

**处理流程**:
```
1. 数据预处理
   ├─ 缺失值填充
   ├─ 类别变量编码
   └─ 数值变量标准化

2. 特征工程
   ├─ 基础特征: 年龄、性别、BMI等
   ├─ 派生特征: 
   │  ├─ 吸烟包年数 = 每天支数 × 吸烟年限 / 20
   │  ├─ 家族史评分 = Σ(患病亲属权重)
   │  └─ 症状风险分 = 症状数量 × 严重程度权重
   └─ 交互特征: 年龄×吸烟, BMI×运动等

3. 模型预测
   ├─ XGBoost模型推理
   ├─ 输出风险概率 (0-1)
   └─ 风险分级 (低/中/高/极高)

4. 可解释性分析
   ├─ SHAP值计算
   ├─ 特征贡献度排序
   └─ 生成解释文本

5. 细分风险评估
   ├─ 肺癌风险（基于吸烟、年龄、症状）
   ├─ 胃癌风险（基于饮食、幽门螺杆菌）
   ├─ 肝癌风险（基于肝炎史、饮酒）
   ├─ 结直肠癌风险（基于饮食、家族史）
   └─ 其他常见肿瘤风险
```

**算法要求**:
- 模型类型: XGBoost / LightGBM
- 准确率: ≥ 85%
- 召回率: ≥ 80%（高风险人群）
- 推理时间: < 1秒

**输出结果**:
```json
{
  "assessmentId": "uuid-xxx",
  "userId": "user-123",
  "overallRisk": {
    "score": 0.68,          // 综合风险分数 (0-1)
    "level": "高风险",       // 风险等级
    "percentile": 82        // 风险百分位（比82%的人高）
  },
  "categoryRisks": {
    "肺癌": {"score": 0.75, "level": "高风险"},
    "胃癌": {"score": 0.45, "level": "中风险"},
    "肝癌": {"score": 0.60, "level": "高风险"},
    "结直肠癌": {"score": 0.50, "level": "中风险"},
    "乳腺癌": {"score": 0.30, "level": "低风险"}
  },
  "keyFactors": [
    {
      "factor": "吸烟史",
      "contribution": 0.35,    // SHAP值
      "direction": "increase", // 增加/降低风险
      "description": "长期吸烟（20年）显著增加肺癌风险"
    },
    {
      "factor": "家族肿瘤史",
      "contribution": 0.22,
      "direction": "increase",
      "description": "直系亲属有肺癌病史"
    },
    {
      "factor": "年龄",
      "contribution": 0.18,
      "direction": "increase",
      "description": "55岁属于肿瘤高发年龄段"
    }
  ],
  "recommendations": [...],  // 见ASSESS-002
  "createdAt": "2024-10-12T10:30:00Z"
}
```

#### ASSESS-002: 个性化健康建议生成

**优先级**: P0（必须）

**功能描述**:  
基于风险评估结果，生成个性化的健康建议和就医指导

**建议类型**:

**1. 生活方式建议**
```
规则引擎示例:
IF 吸烟 == "每天吸烟" AND 肺癌风险 > 0.6:
  建议: "强烈建议戒烟。吸烟是肺癌的首要危险因素，戒烟可显著降低风险。"
  
IF BMI > 28:
  建议: "建议控制体重至正常范围（BMI 18.5-24）。肥胖与多种肿瘤相关。"
  
IF 运动 == "几乎不运动":
  建议: "建议每周至少进行150分钟中等强度有氧运动。"
```

**2. 饮食建议**
```
规则引擎示例:
IF 饮食习惯 CONTAINS "腌制食品":
  建议: "减少腌制、熏制食品摄入，增加新鲜蔬菜水果。"
  
IF 胃癌风险 > 0.5:
  建议: "避免高盐饮食，定期检查幽门螺杆菌，注意胃部健康。"
```

**3. 筛查建议**
```
规则引擎示例:
IF 肺癌风险 > 0.6 AND 年龄 > 50:
```

```sql
CREATE TABLE questionnaires (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    -- 基本信息
    age INTEGER NOT NULL,
    gender VARCHAR(10) NOT NULL,
    height DECIMAL(5,2) NOT NULL,
    weight DECIMAL(5,2) NOT NULL,
    bmi DECIMAL(4,2) GENERATED ALWAYS AS (weight / (height/100)^2) STORED,
    -- 生活习惯（JSON存储详细数据）
    smoking_history JSONB,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**assessments表（风险评估结果）**
```sql
CREATE TABLE assessments (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    questionnaire_id UUID REFERENCES questionnaires(id),
    -- 综合风险
    overall_risk_score DECIMAL(5,4) NOT NULL, -- 0-1
    overall_risk_level VARCHAR(20) NOT NULL, -- 低/中/高/极高
    risk_percentile INTEGER, -- 百分位
    -- 分类风险（JSON）
    category_risks JSONB NOT NULL,
    -- 关键因素（JSON）
```

## 7. AI模型需求

### 7.1 风险预测模型

**模型类型**: XGBoost / LightGBM

**输入特征**（约30-50个）:
```
1. 基础特征:
   - 年龄、性别、BMI

2. 生活习惯特征:
   - 吸烟包年数
   - 饮酒频率编码
   - 运动频率编码
   - 饮食风险评分

3. 疾病史特征:
   - 慢性病数量
   - 家族史评分
   - 手术史标记

4. 症状特征:
   - 症状数量
   - 高危症状标记
   - 症状持续时间

5. 交互特征:
   - 年龄×吸烟
   - BMI×运动
   - 家族史×症状
```

**模型性能要求**:
- 准确率（Accuracy） ≥ 85%
- 召回率（Recall） ≥ 80%（针对高风险人群）
- 精确率（Precision） ≥ 75%
- F1 Score ≥ 0.80
- AUC ≥ 0.88

**推理时间**: < 1秒

**模型文件**:
- `xgboost_model.pkl` (模型权重)
- `scaler.pkl` (标准化器)
- `feature_names.json` (特征列表)

### 7.2 SHAP可解释性

**使用库**: `shap` (Python)

**解释器类型**: TreeExplainer（针对XGBoost）

**输出**:
- 每个样本的SHAP值（向量）
- 特征重要性排序
- 可视化数据（瀑布图、力图）

### 7.3 影像分析模型（可选）

**模型类型**: ResNet50 / EfficientNet-B0

**任务**: 肺结节检测（示例）

**输入**: 224×224 RGB图像

**输出**:
- 分类: 正常 / 可疑结节
- 置信度: 0-1
- 热力图: Grad-CAM可视化

**性能要求**:
- 准确率 ≥ 80%
- 推理时间 < 2秒
