# 📦 安装依赖说明

## 1️⃣ 激活虚拟环境

```bash
# Windows PowerShell
D:\桌面\shensibei\venv\Scripts\Activate.ps1

# 或者 CMD
D:\桌面\shensibei\venv\Scripts\activate.bat
```

## 2️⃣ 安装所有依赖

```bash
# 进入backend目录
cd D:\桌面\shensibei\backend

# 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**说明**：
- 使用清华镜像源加速下载
- 安装时间约 5-10 分钟

## 3️⃣ 验证安装

```bash
# 验证PyTorch（深度学习）
python -c "import torch; print(f'PyTorch: {torch.__version__}')"

# 验证OSS（文件上传）
python -c "import oss2; print(f'OSS2: {oss2.__version__}')"

# 验证Pillow（图像处理）
python -c "from PIL import Image; print('Pillow: OK')"
```

## 4️⃣ 常见问题

### 问题1：pip install 很慢
**解决**：使用国内镜像源（已在命令中）

### 问题2：torch安装失败
**解决**：单独安装
```bash
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

### 问题3：某些包版本冲突
**解决**：升级pip
```bash
python -m pip install --upgrade pip
```

## 5️⃣ 测试OSS连接

创建测试文件 `test_oss.py`：

```python
from app.utils.oss_client import oss_client

# 测试连接
try:
    # 列出bucket（验证权限）
    result = oss_client.bucket.list_objects(max_keys=1)
    print("✅ OSS连接成功！")
    print(f"Bucket: {oss_client.bucket.bucket_name}")
except Exception as e:
    print(f"❌ OSS连接失败: {e}")
```

运行测试：
```bash
python test_oss.py
```
