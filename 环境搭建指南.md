## 🔧 需要安装的软件

### 1. MySQL 8.0+（数据库）

#### Windows安装步骤：
1. 访问：https://dev.mysql.com/downloads/installer/
2. 下载：mysql-installer-community（约400MB）
3. 运行安装程序
4. 选择"Developer Default"（开发者默认）
5. **重要**: 设置root密码（请记住！）
6. 完成安装后，测试：
   ```cmd
   mysql --version
   ```

#### 验证MySQL是否运行：
```cmd
# 登录MySQL
mysql -u root -p
# 输入你设置的密码

# 看到 mysql> 提示符表示成功
# 输入 exit; 退出
```

---

### 2. Redis 7.0+（缓存）

#### Windows安装步骤：
1. 访问：https://github.com/tporadowski/redis/releases
2. 下载：Redis-x64-5.0.14.1.zip（约5MB）
3. 解压到：`C:\Redis`
4. 双击运行：`redis-server.exe`
5. 看到"Ready to accept connections"表示成功

#### 验证Redis是否运行：
```cmd
# 新开一个命令行窗口
cd C:\Redis
redis-cli.exe ping

# 应该返回: PONG
```

## ⬇️ 第4步：安装Python依赖包

```cmd
# 升级pip（推荐）
python -m pip install --upgrade pip

# 安装依赖（可能需要5-10分钟，请耐心等待）
pip install -r requirements.txt
```

```cmd

# 应该显示: fastapi  0.104.1
```

## 🗄️ 第5步：创建MySQL数据库

1. 打开新的命令行窗口
2. 登录MySQL：
   ```cmd
   mysql -u root -p
   ```
3. 输入密码
4. 创建数据库：
   ```sql
   CREATE DATABASE tumor_screening CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
5. 验证：
   ```sql
   SHOW DATABASES;
   ```
   应该能看到 `tumor_screening`
6. 退出：
   ```sql
   exit;
   ```

## 📝 第6步：配置环境变量

在backend目录创建 `.env` 文件（复制以下内容）：

```env
# 数据库配置（修改密码为你的MySQL密码）
DATABASE_URL=mysql+pymysql://root:你的MySQL密码@localhost:3306/tumor_screening
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_NAME=tumor_screening

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_DB=0
REDIS_PASSWORD=

# JWT配置（随机生成32位以上字符）
SECRET_KEY=sk-your-super-secret-key-min-32-chars-long-for-production
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=10080

# 应用配置
APP_NAME=肿瘤数智化筛查系统
APP_VERSION=1.0.0
DEBUG=True
ALLOWED_ORIGINS=http://localhost:5173,http://localhost:3000

# 文件上传
UPLOAD_DIR=uploads
MAX_UPLOAD_SIZE=52428800

# 模型配置
MODEL_DIR=app/ml/models
MODEL_VERSION=v1.0

# 日志
LOG_LEVEL=INFO
LOG_FILE=logs/app.log
```

**重要**: 把 `你的MySQL密码` 替换成实际密码！

## ❓ 常见问题

### Q1: MySQL密码忘了怎么办？

**重置MySQL密码**：
1. 以管理员身份运行CMD
2. 停止MySQL服务
3. 跳过权限启动MySQL
4. 重置密码

### Q2: Redis无法启动？

**解决方案**：
1. 确保端口6379未被占用
2. 检查是否有多个Redis进程
3. 重启电脑后再试

### Q3: pip install很慢？

**使用国内镜像**：
```cmd
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q4: 虚拟环境激活失败？

**PowerShell执行策略问题**：
```powershell
# 以管理员身份运行PowerShell
Set-ExecutionPolicy RemoteSigned

# 然后再激活
venv\Scripts\Activate.ps1
```
